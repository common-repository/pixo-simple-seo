== Description ==

Enjoy!
An small and easy meta data SEO plugin. 
The Plugin insert an title and the description on all pages and articles. 
You also have some generall seetings where you can insert an global title and description, if you don´t want to give every page like 404 individual meta data.

== Installation ==

1. Search my plugin "pixo simple seo" via WordPress plugin internal search.
2. Click on install and then activate.
3. Done

or

1. Upload the unzipped plugin to /wp-content/plugins/ directory via FTP
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enjoy!
